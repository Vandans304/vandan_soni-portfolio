    Here are a few projects I've worked on recently.

    Forum Webiste

    I have created a fully functional dynamic website, featuring a discussion section that facilitates user conversations. For example, when User-1 posts a question, another user can log in later, see the question, and provide an answer. This enables ongoing interactions through questions, answers, and responses.

    Quiz Website

    I have created a fully responsive quiz app using HTML, CSS, and JavaScript, complete with great animations. In this app, users first log in and then are taken to the home page. Before starting the quiz, there is a guide for users to follow. Each quiz has a specific time limit, and after completing the quiz, users can log out of their accounts.

    Travel Agency Website

    I built a travel agency website using HTML and CSS, with form validation for the cart section and other forms. In this project, users first log in to their accounts. The website includes sections like Packages, Services, Gallery, Reviews, and Contact. In the Packages section, users can select their favorite destinations and add them to the cart. The cart section allows users to view and manage their selected packages

    Construction Website

    I created this website to learn CSS. Through this project, I learned CSS media queries for responsiveness, as well as grid, flexbox, animations, positioning, typography, z-index, and more.

    Ecommerce Website

    I created a fully responsive e-commerce website using HTML and CSS, focusing on learning CSS techniques. This project features user login, product sections like Packages, Services, Gallery, Reviews, and Contact, and a cart section for managing selected items. I incorporated CSS media queries, grid, flexbox, animations, positioning, typography, and z-index to enhance the site's design and functionality.

    Food Delivery Website

    I developed and maintained responsive websites using HTML, CSS, and JavaScript, along with modern frameworks like React and Angular. I created interactive animations and transitions to enhance user engagement and improve the visual appeal of the websites. I conducted cross-browser testing and debugging to ensure compatibility and optimize performance. Additionally, I implemented form validations and error handling to improve data integrity and the user experience in web forms
